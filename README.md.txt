# Работа с памятью и массивами в C#

## Описание
Программа демонстрирует работу с различными типами массивов в C# и показывает, как они хранятся в памяти.

## Функциональность
- Создание массивов разных типов (int, double, string)
- Двумерные массивы (матрицы)
- Копирование массивов (глубокая копия)
- Изменение размера массива
- Информация о памяти (размер, адреса)
- Разница между Stack и Heap
